# 업무 관리 칸반보드

HTML · CSS · JavaScript 단일 파일로 만든 업무 관리 칸반보드입니다.  
별도 서버·라이브러리 없이 브라우저에서 바로 실행되며, GitHub Pages로 배포되어 있습니다.

🔗 **Live Demo**: https://hanah9936.github.io/ai_kanban_board/

## 주요 기능

| 기능 | 설명 |
|------|------|
| **3개 컬럼** | TO DO / 진행 중 / 완료 — 컬럼별 색상 및 카드 수 표시 |
| **카드 추가** | 상단 `+` 버튼 또는 하단 `+ 카드 추가` 클릭 |
| **카드 정보** | 제목, 설명, 우선순위(높음/보통/낮음), 마감일, 레이블, 담당자 |
| **드래그 앤 드롭** | 카드를 다른 컬럼으로 자유롭게 이동 |
| **카드 편집/삭제** | 카드에 마우스를 올리면 편집·삭제 버튼 표시 |
| **마감일 경고** | 기한이 지난 카드는 날짜가 빨간색으로 표시 |
| **진행률 표시** | 헤더에 완료 비율 프로그레스 바 실시간 반영 |
| **데이터 유지** | `localStorage` 저장 — 새로고침 후에도 데이터 보존 |

## 로컬 실행 방법

### 파일 직접 열기

```bash
git clone https://github.com/hanah9936/ai_kanban_board.git
cd ai_kanban_board

# macOS
open index.html

# Windows
start index.html
```

### 로컬 서버로 실행

```bash
# Python 3
python -m http.server 8080
# → http://localhost:8080

# Node.js
npx serve .
```

## 사용 방법

1. **카드 추가** — 컬럼 상단 `+` 또는 하단 `+ 카드 추가` 버튼 클릭
2. **정보 입력** — 제목(필수), 설명, 우선순위, 마감일, 레이블, 담당자 입력 후 저장
3. **카드 이동** — 카드를 드래그해 다른 컬럼에 드롭
4. **카드 편집** — 카드 호버 → ✏️ 버튼 클릭
5. **카드 삭제** — 카드 호버 → 🗑 버튼 클릭
6. **단축키** — `Ctrl + Enter` 로 모달 저장 / `Esc` 로 모달 닫기

## 기술 스택

- **HTML5** — 시맨틱 마크업, Drag and Drop API
- **CSS3** — CSS Variables, Grid, Flexbox, 트랜지션 애니메이션

- **localStorage API** — 브라우저 내 데이터 영구 저장
